Introduction to CSS

What is CSS?

1) CSS was developed in 1997 for Web designers, as a way for Web designers to define the attractive look and feel of their Web pages.

2) CSS is used for design web pages.

3) CSS works with HTML only.

4) HTML is only Markup language, we can use some attributes and values in tags, but it's not capable of creating attractive web page design.

5) The World Wide Web Consortium (W3C) has created CSS to solve this problem.

6) In HTML 4.0, all formatting could be removed from the HTML document and can be stored in a separate CSS file.

7) Today all web browsers are supporting CSS.
